# 🤖 AI Personal Assistant (Flask + Groq)
<img width="1883" height="893" alt="Screenshot 2026-04-14 200442" src="https://github.com/user-attachments/assets/855be82d-eaff-4c36-945b-0f27b6f69291" />

A simple AI-powered web application built using Flask that can:
- Answer user queries 💬
- Summarize emails ✉️  
Powered by Groq LLM API.


## Features
- Ask anything (Q&A chatbot)
- Email summarization (2–3 line summaries)
- Fast responses using LLM

## Tech Stack
- Python (Flask)
- Groq API (LLM)
- HTML/CSS/Javascript 


## Setup

1. Clone the repo:
git clone https://github.com/ishu2293/AI_Personal_Assistant.git
cd AI_Personal_Assistant

2. Install dependencies:
pip install -r requirements.txt

3. Create .env file:
API_Key_GROQ=your_api_key_here

4. Run the app:
python main.py
